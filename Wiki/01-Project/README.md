# Forge Wiki

This is the living brain for the **Forge** repository: a lean software delivery framework for AI agents.

---

## What is Forge?

Forge defines 21 skills across 7 categories, a Linear-backed state machine, and a "perfect loop" architecture. The goal is to turn a collection of process documents into a coherent loop operating system.
